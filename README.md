# siyuan

An SNS backend framework, with Node and MySQL

## Install

1. Make sure you have Node and MySQL installed already

2. Download the package, and enter it
	```shell
	cd siyuan
	```

3. Install the dependencies
	```shell
	npm install -d
	```

4. Setup the database
	```shell
	node setup
	```

## Run

```shell
npm start
```

## Test

```shell
npm test
```
